# Introduction

**Mep** (MepCLI) is a comprehensive, zero-dependency, type-safe interactive CLI prompt library for Node.js, Deno, and Bun. It provides a vast collection of customizable prompts ranging from simple text inputs to complex data visualization and rich UI components.

## Key Features

- **Zero Dependencies**: Lightweight and fast, with no external runtime dependencies.
- **Type-Safe**: Built with TypeScript, offering full type inference for prompt results.
- **Rich Component Library**: Includes over 50 prompt types (Select, Date, Spreadsheet, Kanban, etc.).
- **Themable**: easy customization of colors and styles.
- **Utilities**: Built-in Task Runner, Spinner, and Pipeline support.
- **Cross-Platform**: Works on Windows, macOS, and Linux.

## Why Mep?

Unlike other libraries that focus on a few basic prompts, Mep aims to be a complete toolkit for building terminal user interfaces (TUIs). whether you need a simple confirmation or a full-blown dashboard with charts and tables, Mep has a component for it.
